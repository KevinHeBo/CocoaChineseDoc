Draw 2D and 3D content and animate your application's user interface with the unsurpassed graphics and animation technologies of iOS—Quartz, Core Animation, and OpenGL ES. Quartz is a powerful 2D drawing engine for creating vector graphics, bitmap images, and PDF content. Core Animation is the technology that adds smooth motion and dynamic feedback to your user interface by creating an illusion of motion. OpenGL ES is the interface for high-performance 3D graphics on mobile devices.
