iOS Starting Point

iOS comprises the operating system and technologies you use to create applications for iOS-based devices. The technologies in iOS provide everything you need to create advanced applications that respond to touch events and display high-quality graphics. In addition to the basic application infrastructure, iOS gives you access to multitouch events, the hardware accelerometers, and other device features that make it easy to create advanced and innovative applications.

The iOS SDK includes iOS, the Xcode tools, and access to the documentation, sample code, and resources you need to develop your applications. The iOS SDK and Xcode tools can be used on any Intel-based Macintosh computer with a current version of Mac OS X. (For information about specific Mac OS X requirements, see the release notes for the version of the iOS SDK you are using.) Xcode provides the development environment you use to create your application code, compile it, run it, and debug it. Get Up and Running

Learn the basics about iOS development by reading the following documents:

  • iOS Overview is a high-level introduction to iOS and its technologies.

  • Tools for iOS Development provides an overview of the tools you use to create iOS applications.

  • Learning Objective-C: A Primer introduces the Objective-C programming language used for iOS development. The audience for this document is existing C and C++ developers who want a simple introduction to the Objective-C language so that they can start writing code.

  • Creating an iPhone Application gives you a guided tour through the code of the MoveMe sample application, which displays custom content and responds to user interactions with that content.

Become Proficient

Read the following documents in the early stages of your product development. They contain detailed information essential to all iOS developers.

  • iOS Development Guide describes the process for setting up your computer and iOS-based devices to support iOS development.

  • iOS Application Programming Guide describes the fundamental structure and features of an iOS application and shows you how to implement those features in your own projects.

  • Your First iOS Application steps you through the creation of a simple iOS application, explaining the tools and techniques you use for iOS development along the way.

  • iOS Human Interface Guidelines provides guidance for designing an excellent user interface and user experience appropriate for the iOS platform.

  • iOS Technology Overview describes the high-level architecture and technologies available in iOS.

  • Cocoa Fundamentals Guide provides detailed information about the programming conventions and design methodologies used to create iOS applications.

  • The Objective-C Programming Language provides an in-depth guide to the Objective-C language and runtime.
